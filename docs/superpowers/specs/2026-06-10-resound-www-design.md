# Resound Landing Page

## Overview
A Next.js landing page at `resound-www/` for the Resound macOS app. Dark/emerald aesthetic matching the app's visual identity.

## Aesthetic
- **Name:** Dark Emerald Editorial
- **Colors:** Deep charcoal (#0A0A0A) backgrounds, emerald (#0D5E3C) accents, white text with opacity layers
- **Fonts:** Bricolage Grotesque (display), Sora (body)
- **Memo anchor:** The Dynamic Island pill shape as a recurring decorative element

## Sections
1. **Hero** — Full-screen dark, pill mockup at top of screen, tagline, CTA button
2. **Features** — 3×2 grid with icons, "Coming Soon" badges on unshipped features
3. **Install** — Three numbered steps, code block with `sudo xattr` command
4. **FAQ** — Accordion-style Q&A
5. **Footer** — "Built by Evans Elabo", GitHub link

## Tech Stack
- Next.js 14+ (App Router)
- Tailwind CSS
- Google Fonts (Bricolage Grotesque + Sora)
- No external UI libraries

## Features Grid
- Dynamic Island Pill (shipped)
- Waveform Visualization (shipped)
- Global Hotkeys (shipped)
- Spotify & Apple Music (shipped)
- Menu Bar Controls (shipped)
- Customizable Position (coming soon)
